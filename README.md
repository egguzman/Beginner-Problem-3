This is a friendly beginner program. It calculates the rectangle area using the L * W = A formula. 

The program ask the user to enter the rectangle length and then ask to enter the width of the rectangle. 
Once the user is done entering the numbers, the program solves it and gives the area of the rectangle.

        Enter the length of the rectangle: 27
        Enter the width of the rectangle: 11
        The area of the rectangle is: 297
